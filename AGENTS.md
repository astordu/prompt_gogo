## Agent skills

### Issue tracker

Issue 以 GitHub Issues 形式存放在 astordu/prompt_gogo，使用 `gh` CLI 操作。See `docs/agents/issue-tracker.md`.

### Triage labels

使用默认的五个规范分诊标签（needs-triage / needs-info / ready-for-agent / ready-for-human / wontfix）。See `docs/agents/triage-labels.md`.

### Domain docs

单上下文布局——根目录 `CONTEXT.md` + `docs/adr/`。See `docs/agents/domain.md`.
