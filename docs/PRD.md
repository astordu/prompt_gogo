# Prompt Go 桌面程序需求文档

## 项目概述
开发一款类似 Prompt Go 的桌面应用程序，用于快速处理选中文本。

## 技术栈
- **开发框架**：Electron
- **平台支持**：Mac 版本（第一版 MVP）
- **AI 模型**：DeepSeek

## 核心功能

### 1. 配置页面
提供统一的设置界面，包含以下配置项：
- **API Key 配置**：输入 DeepSeek API Key
- **快捷键管理**：支持配置多个快捷键及其对应的提示词模板

### 2. 快捷键与提示词模板
- 每个快捷键对应一个提示词模板
- 支持配置多组"快捷键-模板"映射关系
- **重要约定**：所有提示词模板必须包含 `{{select_content}}` 字段，用于接收用户选中的文本

### 3. 文本处理流程
1. 用户在任意应用中选中文本
2. 按下配置好的快捷键
3. 程序自动调用对应的提示词模板
4. 将选中文本代入模板的 `{{select_content}}` 位置
5. 调用 DeepSeek 进行处理
6. 返回重新梳理后的内容

## 使用场景示例
用户选中一段文字后，通过快捷键触发不同的处理模板，例如：
- 快捷键 1：格式化整理
- 快捷键 2：翻译
- 快捷键 3：总结提炼
