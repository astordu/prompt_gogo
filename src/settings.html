为了让 Prompt Go 正常工作，您需要授予以下权限：

1

辅助功能权限

此权限用于读取和操作选中的文本内容。

系统设置 → 隐私与安全性 → 辅助功能

找到 Prompt Go，勾选启用

2

屏幕录制权限（可选）

某些情况下可能需要此权限来获取屏幕内容。

系统设置 → 隐私与安全性 → 屏幕录制

找到 Prompt Go，勾选启用

lightbulb

💡 提示

如果权限设置后仍无法使用，请尝试：
1. 完全退出 Prompt Go 后重新打开
2. 在权限设置中移除 Prompt Go，然后重新添加
3. 重启您的 Mac
